Some Stuff We Learned

  1. HTML code
    1. All text must be within a tag or it is inaccesable/unmodifiable
    2. The semantic tags clean up the look of the code making it easier to understand and better for CSS
    3. Having clear thought out code and comments makes it easier for you and others to understand the intent of the code
  2. CSS
    1. CSS follows certain(if/then) rules using selecyors and declarations
    2. CSS declarations can have multiple selectors attatched to them and multiple of both per tag
    3. Colors in CSS can be written as text,RGB, or hexadecimal
  3. Elements
    1. These are the tags that we use to create html webpages, including what you're reading right now
    2. Elements can be block line meaning they always start on a new line and take up the full space available
    3. Or they can be inline meaning they don't start on a new line nor do they take up the full space available
